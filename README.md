# REST-API-Nodejs

### How to use

- Clone This Repository
- Got to project directory
- NPM install
-Create your own env file

#### API list
                
1. GET ALL user (/users)
2. GET Single user(/users/:id)
3. DELETE (users/:id)
4. PATCH (users/:id)
